# DNS Resolution

# Zones and queries

  • Zone: Collection of domain names.
  • Delegation: Domain name in a zone delegated to have information about child nodes or underlying sub domains.

  • DNS queries:
  • Recursive query: When a client ( computer) demands an answer (IP address) from a local DNS server (ISP DNS).
  • Iterative query: When a local DNS server cache, doesn't have a record for the recursive query. It asks other DNS servers for referals or an answer.
